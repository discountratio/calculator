CALCULATOR 2000

Calculator

The calculator project was an excellent way to get used to DOM manipulation and JavaScript fundamentals. The calculator is styled using CSS grid for the layout and googlefonts for the text. Each button has a 'click' event listener, and there is a switch statement to assess button type for the appropriate action. The calculator is reasonably functional, there are some minor issues that exist for the sake of design (i.e. calculation inputs / outputs are limited to 8 digits to prevent overflow).

Possible improvements I would like to make in the future include: responsiveness, calculation history, and key press effects replaced with animations

An external library, VanillaTilt.js was used for the tilt and glare effects.
